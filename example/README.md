This example package demonstrates three biology-themed ClickQt applications.

Install in editable mode:

```bash
python -m pip install --editable ./example
```

Launch the suite launcher:

```bash
example_cli
example_gui
```

Launch individual apps (CLI):

```bash
example_bio_hello_cli --help
example_bio_pcr_cli --help
example_bio_rnaseq_cli --help
```

Launch individual apps (GUI):

```bash
example_bio_hello_gui
example_bio_pcr_gui
example_bio_rnaseq_gui
```
